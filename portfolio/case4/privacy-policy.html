1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей сайта bureau-of-talents.ru (далее — «Сайт»), принадлежащего Индивидуальному предпринимателю Виктории (далее — «Оператор»).

Используя Сайт, заполняя формы обратной связи и загрузки материалов, Пользователь выражает своё согласие с условиями настоящей Политики. Если Пользователь не согласен с условиями, он обязан прекратить использование Сайта.

Настоящая Политика действует в отношении всех персональных данных, которые Оператор может получить о Пользователе во время использования им Сайта.

Обработка персональных данных осуществляется в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

2. Перечень собираемых персональных данных

Оператор собирает следующие данные Пользователя:

  • Имя — указывается Пользователем в формах обратной связи и загрузки гайда;
  • Адрес электронной почты (e-mail) — указывается Пользователем в формах;
  • Номер телефона — указывается Пользователем в формах (по желанию);
  • Наименование компании — указывается Пользователем в форме загрузки гайда (по желанию);
  • Текст сообщения — указывается Пользователем в форме обратной связи (по желанию);
  • Данные cookie — техническая информация, автоматически передаваемая Сайтом в процессе его использования с помощью установленного на устройстве Пользователя программного обеспечения (подробнее в разделе 8).

3. Цели обработки персональных данных

Персональные данные Пользователя обрабатываются в следующих целях:

  • Обработка запросов, направленных через формы обратной связи на Сайте;
  • Предоставление Пользователю запрошенных материалов (в том числе гайда «Как написать вакансию, которая нанимает без вас»);
  • Связь с Пользователем для уточнения деталей запроса, консультирования по услугам;
  • Улучшение качества работы Сайта, анализ поведения Пользователей с помощью cookie;
  • Направление информационных сообщений (только при отдельном согласии Пользователя).

4. Правовые основания обработки персональных данных

Правовыми основаниями обработки персональных данных являются:

  • Федеральный закон от 27.07.2006 № 152-ФЗ «О персональных данных»;
  • Согласие Пользователя на обработку персональных данных (выражается путём отметки чекбокса в формах Сайта);
  • Согласие Пользователя на обработку cookie-файлов (выражается через нажатие кнопки «Принять» в cookie-баннере).

5. Сроки хранения персональных данных

Персональные данные Пользователя хранятся не дольше, чем этого требуют цели их обработки, если иное не предусмотрено законодательством РФ.

  • Данные, полученные через формы обратной связи и загрузки гайда, хранятся в течение 1 года с момента получения запроса;
  • Данные cookie хранятся в течение срока действия сессии браузера или до момента очистки кэша и cookie Пользователем;
  • По истечении сроков хранения данные подлежат удалению или обезличиванию.

6. Порядок передачи персональных данных третьим лицам

Оператор не передаёт персональные данные Пользователей третьим лицам, за исключением следующих случаев:

  • Обработка данных осуществляется с помощью почтового сервиса EmailJS для отправки материалов Пользователю и уведомлений Оператору. EmailJS обрабатывает данные исключительно в объёме, необходимом для оказания услуги, и обязуется соблюдать конфиденциальность в соответствии со своей Политикой конфиденциальности;
  • Предоставление данных по запросу уполномоченных органов государственной власти РФ в случаях, предусмотренных законодательством;
  • В иных случаях — только с письменного согласия Пользователя.

Оператор не осуществляет трансграничную передачу персональных данных.

8. Использование cookie-файлов

Сайт использует файлы cookie — небольшие текстовые файлы, которые сохраняются на устройстве Пользователя при посещении Сайта. Cookie позволяют:

  • Обеспечить корректную работу Сайта (функциональные cookie);
  • Собирать анонимную статистику о посещениях и поведении Пользователей для улучшения работы Сайта (аналитические cookie).

Пользователь даёт согласие на обработку cookie-файлов путём нажатия кнопки «Принять» в cookie-баннере, который отображается при первом посещении Сайта. Пользователь может отключить cookie в настройках своего браузера. Отключение cookie может привести к некорректной работе некоторых функций Сайта.
